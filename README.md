# 🇸🇦 Saudi Vision 2030 Landing Page

A beautiful, high-performance, and bilingual (English/Arabic) landing page celebrating **Saudi Vision 2030**. 

Developed with ❤️ by [Ohoud Alawad (عهود العواد)](https://www.ohoudalawad.dev/).

## 🚀 Key Features

- **Dual-Language Support**: Complete translations for English (LTR) and Arabic (RTL) with dynamic layout direction toggling and context-based font loading (*Inter* & *Outfit* for English, *Tajawal* for Arabic).
- **Premium Design System**: Engineered with Tailwind CSS v4.0 and customized with a majestic Emerald Green and Sand Gold palette, glassmorphism panels, and smooth scroll entry animations.
- **Core Web Vitals Optimized**: Zero layout shifts (CLS), lazy-loaded project assets, and Next.js Image component preloading (`priority`) for the Largest Contentful Paint (LCP) banner.
- **SEO Ready**: Configured metadata base, OpenGraph social properties, sitemaps, and robots directives for optimized indexing and search discovery.
- **Responsive Layout**: Works beautifully across mobile, tablet, and desktop viewports.

## 🛠️ Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS v4.0
- **Language**: TypeScript
- **Icons**: Lucide React & custom inline SVGs

## 📦 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🏗️ Production Build

To build and optimize the application for production, compile the static site by running:

```bash
npm run build
```
